## Requirements

- mplayer
- gobot.io/x/gobot 
- gobot.io/x/gobot/platforms/dji/tello

```sh
$ brew install mplayer
$ go get gobot.io/x/gobot 
$ go get gobot.io/x/gobot/platforms/dji/tello
```
